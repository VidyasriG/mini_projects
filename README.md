# Python Mini Projects

This repository contains beginner-friendly Python applications demonstrating basic programming concepts.

## Projects Included

### 1. Calculation Program
- Performs basic calculations
- Handles user input and arithmetic operations

### 2. To-Do List Application
- Add new tasks
- View tasks
- Simple task management system

## Technologies Used
- Python

## Concepts Covered
- If-else conditions
- Functions
- User input/output
- Basic data structures

##  How to Run

1. Clone the repository:
   git clone https://github.com/VidyasriG/mini_projects.git

2. Navigate to the folder:
   cd mini_projects

3. Run the program:
   python app.py

## Purpose
This project is created to practice Python fundamentals and build simple real-world applications.
